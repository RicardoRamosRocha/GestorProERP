# Veltis ERP — Roadmap do Produto

## Objetivo

Este roadmap apresenta a evolução planejada do Veltis ERP, definindo as entregas previstas para cada fase do projeto.

---

# Fase 1 — Planejamento

## Objetivos

* Definir visão do produto
* Levantar requisitos
* Criar backlog
* Definir casos de uso
* Planejar sprints
* Modelar banco de dados
* Definir arquitetura da solução

## Entregáveis

* Documento de visão
* Documento de requisitos
* Backlog do produto
* Casos de uso
* Roadmap
* Arquitetura inicial
* Modelo de banco de dados

---

# Fase 2 — Fundação da Solução

## Objetivos

* Criar solução ASP.NET Core
* Configurar SQL Server
* Configurar Entity Framework Core
* Configurar Git e GitHub
* Estruturar arquitetura em camadas

## Entregáveis

* Solução criada
* Banco configurado
* Migrations configuradas
* Repositório GitHub estruturado

---
